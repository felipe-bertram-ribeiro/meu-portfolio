Bookstore MVC

Bookstore management system built with ASP.NET MVC and SQL Server.

A complete project for managing books, clients, and loans, featuring an intuitive dashboard and responsive design.

About the Project

Bookstore MVC is a system developed to manage all bookstore operations — from client and book registration to loan control and performance reports. The application follows the MVC (Model-View-Controller) pattern and uses a SQL Server database.

Main Features:

  • Client, book, and loan management
  • Dashboard with statistics and visual indicators
  • Advanced search and filtering
  • Real-time detailed reports
  • Intuitive and responsive Bootstrap interface

Technologies Used

C# ASP.NET MVC SQL Server Bootstrap JavaScript HTML CSS
