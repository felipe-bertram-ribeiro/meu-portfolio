Calcular Comprimento das Vias - Avançado

Plugin QGIS para cálculo de comprimento total e por categoria de vias lineares.

Permite calcular o comprimento total das feições, criar campos adicionais com valores individuais e exportar resultados em Shapefile ou CSV.

Sobre o Projeto

O Calcular Comprimento das Vias - Avançado é um plugin para QGIS desenvolvido em Python que automatiza o cálculo de comprimento de feições lineares. Ele permite:

  • Calcular o comprimento total de todas as feições de uma camada;
  • Realizar cálculos agrupados por categoria do campo highway;
  • Criar automaticamente o campo comp_km contendo o comprimento de cada feição;
  • Exportar resultados em Shapefile reprojetado (EPSG:31982) ou CSV.

Os cálculos são feitos no sistema de coordenadas EPSG:31982, garantindo precisão métrica mesmo que a camada original esteja em outro CRS.

Funcionalidades principais:

  • Seleção de camadas vetoriais do tipo linha
  • Cálculo do comprimento total geral em quilômetros
  • Cálculo do comprimento total por categoria do campo highway
  • Criação automática do campo comp_km em cada feição
  • Exportação para Shapefile reprojetado (EPSG:31982)
  • Exportação de resultados em CSV

Por que criei esse plugin

No meu trabalho, precisávamos calcular manualmente o comprimento de vias lineares e agrupar por categoria, o que era demorado e sujeito a erros. Para automatizar esse processo, criei o Calcular Comprimento das Vias, permitindo:

  • Selecionar a camada desejada diretamente no QGIS;
  • Escolher entre cálculo total ou por categoria;
  • Gerar campo adicional com comprimento individual de cada feição;
  • Exportar resultados facilmente em CSV ou Shapefile.

Tecnologias Usadas

Python QGIS PyQt5 Shapefile / Camadas Vetoriais CRS EPSG:31982

Como Usar

  1. Instale o plugin copiando a pasta para a pasta de plugins do QGIS ou usando o repositório GitHub.
  2. Abra o QGIS e carregue a camada vetorial contendo feições lineares.
  3. Abra o plugin pelo menu Calcular Comprimento das Vias.
  4. Selecione a camada desejada e escolha entre Calcular Total Geral ou Calcular por Categoria.
  5. Visualize os resultados na área de texto exibida na janela do plugin.
  6. Se desejar, exporte a camada resultante em Shapefile ou CSV.

Informações Técnicas

  • Nome do Plugin: Calcular Comprimento Vias
  • Versão: 1.0.0
  • Compatibilidade: QGIS 3.x ou superior
  • Campo criado: comp_km
  • CRS utilizado: EPSG:31982
