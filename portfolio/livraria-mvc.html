Livraria MVC

Sistema de gerenciamento de livraria com ASP.NET MVC e SQL Server.

Um projeto completo para controle de livros, clientes e empréstimos, com dashboard intuitivo e design responsivo.

Sobre o Projeto

O Livraria MVC é um sistema desenvolvido para gerenciar todas as operações de uma livraria, desde o cadastro de clientes e livros até a gestão de empréstimos e relatórios de desempenho. A aplicação segue o padrão MVC (Model-View-Controller) e utiliza banco de dados SQL Server.

Funcionalidades principais:

  • Cadastro de clientes, livros e empréstimos
  • Dashboard com estatísticas e indicadores visuais
  • Busca e filtros avançados
  • Relatórios detalhados em tempo real
  • Interface intuitiva e responsiva com Bootstrap

Tecnologias Usadas

C# ASP.NET MVC SQL Server Bootstrap JavaScript HTML CSS
